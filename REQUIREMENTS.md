## Step 1 기능 목록

- [x] 새로운 카드 등록
- [x] 카드 입력 폼
  - [x] 카드 번호
  - [x] 만료일 (MM / YY)
  - [x] 카드 소유자 이름(선택), 30자 제한
  - [x] 보안 코드 (CVC / CVV)
  - [x] 카드 비밀번호, 앞 두자리

## Step 2 기능 목록

- [x] 카드 등록 시 카드사를 선택할 수 있다.
  - [x] 카드사 선택 모달
  - [x] 카드 미리보기에서 카드사, 카드색 변경
- [x] 카드 등록 완료 후 카드 이름을 등록할 수 있다.
- [x] 카드 정보가 없는 상태에서 카드 이름 등록 페이지에 접근할 수 없다.
